Sticky footer

qua 03 julho 2019

Get the footer to stick to the bottom of the window even when there is not enough content to fill the page.

HTML:

<body>
  <header>…</header>
  <main>…</main>
  <footer>…</footer>
</body>

CSS:

body {
  display: flex;
  min-height: 100vh;
  flex-direction: column;
}

main {
  flex: 1;
}
