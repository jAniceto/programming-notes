Hosting Python scripts and services

Thu 07 September 2017, last updated Mon 29 January 2018

Hosting options:

  • Google App Engine (Free Tier)
  • Amazon EC2 (Free Tier)
  • Python Anywhaere (Free Tier)
  • Heroku (Free Tier disconects when inactive, 7$/month Min)
  • Digital Ocean (5$/month Min, 50$ credit from Github Student Pack)
  • Linode (5$/month Min)

Useful Software

  • PuTTY - for SSH access
  • WinSCP - To transfer files to the server
  • FileZilla - An alternative to WinSCP
  • git - For pulling code from remote repository
