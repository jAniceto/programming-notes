# Editing photos metadata

You can do it in one line in the Terminal using `jhead`.

For example, adjust all EXIF times forward by 1 hour:
```
jhead -ta+1:00 *.jpg
```

Useful links: 

[jhead download page](http://www.sentex.net/~mwandel/jhead/)
